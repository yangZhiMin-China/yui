Notify 通知
基础用法
5s后自动关闭 10s后关闭 不自动关闭 隐藏关闭图标 

        <a onclick="yuiNotifyOpen()" yui-button>5s后自动关闭</a>

        <a onclick="yuiNotify('提示','这是一条通知，打开后10s关闭',{delay:10000})" yui-button>
            10s后关闭
        </a>

        <a onclick="yuiNotify('提示','这是一条通知，不会自动关闭',{delay:0})" yui-button>
            不自动关闭
        </a>

        <a onclick="yuiNotify('提示','这是一条通知，不显示关闭图标',{showCloseIcon:false})" yui-button>
            隐藏关闭图标
        </a>

        <script>

            function yuiNotifyOpen() {
                yuiNotify('提示', '这是一条通知，关闭前有回调函数', {
                    beforeClose: function (name, callback) {
                        yuiMessage(`Notify关闭前的回调`, {type: 'success', delay: 1000});
                        callback()
                    }
                });
            }

        </script>
带有倾向性
消 息 成 功 警 告 危 险 

        <a onclick="yuiNotify('提示','这是一条消息通知',{type:'info'})" yui-button type="info">
            消 息
        </a>

        <a onclick="yuiNotify('提示','这是一条成功通知',{type:'success'})" yui-button type="success">
            成 功
        </a>

        <a onclick="yuiNotify('提示','这是一条警告通知',{type:'warning'})" yui-button type="warning">
            警 告
        </a>

        <a onclick="yuiNotify('提示','这是一条危险通知',{type:'danger'})" yui-button type="danger">
            危 险
        </a>
事件
方法 说明 参数 返回
yuiNotify(title,content,options) 打开Message title：标题
content：内容
options：配置，具体见下表 		生成的Notify名称
yuiNotify中的options
参数 说明 类型 可选值 / 回调参数 默认值 / 参数形式
type 类型 string info / success / warning / danger --
delay 延迟关闭时间，如果为0则不自动关闭 number：单位毫秒(ms) -- 5000
showCloseIcon 是否显示关闭图标 boolean true / false true
beforeClose 自动关闭或者手动关闭的回调，，可以使用 callback() 或者 return true 进行关闭 function name:Notify名称
callback:关闭Notify的方法 		(name,callback)
