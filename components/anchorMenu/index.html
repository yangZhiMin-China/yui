yui-anchor-menu
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
click-top 点击锚点时对应模块跳转到的位置距离顶部的大小 number：数字（px）
percent：视窗高度的百分数		 -- 25%
scroll-top 模块滚动到距离顶部多少时相应的锚点变成选中状态 number：数字（px）
percent：视窗高度的百分数		 -- 50%
fixed-top 菜单滚动到距离顶部多少时进行fixed固定 number：数字（px）
percent：视窗高度的百分数		 -- 0
fixed-class 菜单的位置为fixed时自定义class类名 string -- --
yui-click 点击锚点时的回调，如果返回false，则阻止模块的滚动 functionName：方法名 value：被点击的锚点value值 (value)
