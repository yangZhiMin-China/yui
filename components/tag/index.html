yui-tag
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
type 类型 string primary / info / success / warning / danger primary
theme 主题 string light / dark / plain light
size 标签尺寸 string large / small / mini --
padding 设置标签内边距改变标签尺寸 原生style属性 -- 0 10px
width 设置标签宽度 原生style属性 -- --
height 设置标签高度 原生style属性 -- --
line-height 设置标签文字高度 原生style属性 -- 30px
