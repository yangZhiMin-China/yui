yui-button
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
type 类型 string primary / success / warning / danger / info --
plain 朴素按钮样式 是否具有该属性 -- --
radius 圆角按钮样式 是否具有该属性 -- --
text 文字按钮样式 是否具有该属性 -- --
disabled 按钮禁用 是否具有该属性 -- --
size 按钮尺寸 string large / small / mini --
padding 设置按钮内边距改变按钮尺寸 原生style属性 -- 10px 18px
width 设置按钮宽度 原生style属性 -- --
height 设置按钮高度 原生style属性 -- --
