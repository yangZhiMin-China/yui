yui-anchor
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
click-top 点击锚点时对应模块跳转到的位置距离顶部的大小 number：数字（px）
percent：视窗高度的百分数		 -- 25%
scroll-top 模块滚动到距离顶部多少时相应的锚点变成选中状态 number：数字（px）
percent：视窗高度的百分数		 -- 50%
align-center 锚点菜单位置是否需要垂直居中 是否具有该属性 -- --
top 菜单距离上边的偏移量（fixed定位） 原生style属性 -- 0px
left 菜单距离左边的偏移量（fixed定位） 原生style属性 -- --
bottom 菜单距离下边的偏移量（fixed定位）（如果设置了top，那么该属性无效） 原生style属性 -- --
right 菜单距离右边的偏移量（fixed定位）（如果设置了left，那么该属性无效） 原生style属性 -- 0px
yui-click 点击锚点时的回调，如果返回false，则阻止模块的滚动 functionName：方法名 name：被点击的锚点名称 (name)
yui-anchor > div
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
value 锚点的值 string -- --
checked 锚点选中状态 是否具有该属性 -- --
yui-anchor-module
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
yui-anchor-module 标识为锚点模块，与锚点的值对应 string -- --
