Radio 单选框
基础样式
北京 上海 福建厦门 深圳 

        <div yui-click="radioClick" yui-radio">
            <a value="BJ" checked>北京</a>
            <a value="SH" disabled>上海</a>
            <a value="XM">福建厦门</a>
            <a value="SZ">深圳</a>
        </div>

        <script>

            function radioClick(value) {
                if (value === "SZ") {
                    yuiMessage(`拒绝选中 ${value}`, {type: "danger", delay: 1000});
                    return false
                }
                yuiMessage(`选中 ${value}`, {type: "success", delay: 1000});
            }

        </script>

yui-radio
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
type 类型 string button --
size 单选按钮尺寸（type="button"时有效） string large / small / mini --
width 设置radio宽度 原生style属性 -- --
height 设置radio高度 原生style属性 -- --
yui-click 点击radio时的回调，return false则阻止选中 functionName：方法名 value：被点击的radio的value值 (value)
yui-radio > a
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
value radio的值 string -- --
checked radio选中状态 是否具有该属性 -- --
disabled radio禁用状态 是否具有该属性 -- --
