yui-ribbon
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
type 类型 string primary / info / success / warning / danger primary
position 位置 string top-left / top-right / bottom-left / bottom-right top-right
background-color 背景颜色 原生style属性 -- #409eff
border-color 边框颜色 原生style属性 -- --
color 文字颜色 原生style属性 -- #fff
