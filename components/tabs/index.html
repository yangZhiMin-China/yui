yui-tabs
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
tab-position 选项卡位置 string top / bottom / left / right top
type 选项卡类型 string card / border-card --
yui-click 点击选项卡时的回调， return false则阻止选项卡的切换 functionName：方法名 value：被点击的选项卡的value值 (value)
yui-tabs > yui-header > a
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
value 选项卡的值 是否具有该属性 -- --
checked 选项卡选中状态 是否具有该属性 -- --
disabled 选项卡禁用状态 是否具有该属性 -- --
yui-tab-panel
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
yui-tab-panel 选项卡对应的面板，与选项卡的value对应 string -- --
checked 选项卡面板选中（显示）状态 是否具有该属性 -- --
