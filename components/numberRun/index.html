NumberRun 数字滚动
支持小数
开 始 1995.05 

        <a type="primary" yui-button size="mini" onclick="num1Run()">
            开 始
        </a>
        <span id="num1">1995.05</span>

        <script>

            function num1Run() {
                yuiNumberRun('num1', 1995.05)
            }

        </script>
定义滚动次数和滚动完成后的回调
开 始 1995.05 

        <a type="primary" yui-button size="mini" onclick="num2Run()">
            开 始
        </a>
        <span id="num2">1995.05</span>
        <script>

            function num2Run() {
                yuiNumberRun('num2', 2020, {
                    callback: function (number) {
                        yuiMessage(`数字 ${number} 滚动完成`, {type: "success", delay: 1000})
                    },
                    count: 100
                })
            }

        </script>
事件
方法 说明 参数 返回
yuiNumberRun(id, number, options) 开始数字滚动 id：数字显示的元素id名称
number：最后要显示的数字
options：配置，具体见下表 		--
yuiNumberRun中的options
参数 说明 类型 可选值 / 回调参数 默认值 / 参数形式
callback 数字滚动完成后的回调方法 function number:滚动的数字 (number)
count 每位数字的滚动次数 number -- 20
