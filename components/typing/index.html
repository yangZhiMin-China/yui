Typing 打字机
开 始 重 置 

        <div yui-typing id="typing-demo"></div>
        <a yui-button type="primary" onclick="start()">开 始</a>
        <a yui-button onclick="reset()">重 置</a>

        <script>

            function start() {
                yuiTyping("typing-demo", [
                    '武汉市疾病预防控制中心专家金小毛介绍，根据《中华人民共和国传染病防治法》（下称“《防治法》”）规定，传染病分为甲类、乙类和丙类。',
                    '<br/>', '<br/>',
                    '甲类传染病是指：鼠疫、霍乱；',
                    '<br/>', '<br/>',
                    '乙类传染病是指：传染性非典型肺炎、艾滋病、病毒性肝炎、脊髓灰质炎、人感染高致病性禽流感等；'
                ], {
                    delay: 70,
                    callback: function () {
                        yuiMessage(`Typing完成后的回调`, {type: "success"})
                    }
                })
            }

            function reset() {
                yuiTyping("typing-demo")
            }

        </script>

yui-typing
属性 说明 类型 可选值 / 回调参数 默认值 / 参数形式
yui-typing 该属性可给容器加上光标 是否具有该属性 -- --
事件
方法 说明 参数 返回
yuiTyping(id,textArray,options) 开始打印 id：元素ID
textArray：文字内容数组
options：配置，具体见下表 		--
yuiTyping中的options
参数 说明 类型 可选值 / 回调参数 默认值 / 参数形式
delay 打印一个字的延迟时间 number：单位毫秒(ms) -- 100
callback 打印完成后的回调 function -- --
