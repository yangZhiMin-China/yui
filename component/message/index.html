Message 消息提示
常用于主动操作后的反馈提示。与 notify 的区别是后者更多用于系统级通知的被动提醒。
基础用法
从顶部出现，3 秒后自动消失。
打开消息提示
不同状态
用来显示「成功、警告、消息、错误」类的操作反馈。

主题
Message 组件提供了两个不同的主题：light和dark。 通过设置effect属性来改变主题，默认为light。
