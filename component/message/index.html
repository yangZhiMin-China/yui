Message 消息提示
常用于主动操作后的反馈提示。
基础用法
消息 成功 警告 危险 

        <a onclick="yuiMessage('这是一条消息提示')" yui-button>消息</a>

        <a onclick="yui.message.success('恭喜你，这是一条成功消息')" yui-button>成功</a>

        <a onclick="yuiMessage('警告哦，这是一条警告消息',{type:'warning'})" yui-button>警告</a>

        <a onclick="yui.message.danger('错了哦，这是一条错误消息')" yui-button>危险</a>
深色
消息 成功 警告 危险 

        <a onclick="yuiMessage('这是一条消息提示',{effect:'dark'})" yui-button>消息</a>

        <a onclick="yui.message.success('恭喜你，这是一条成功消息',{effect:'dark'})" yui-button>成功</a>

        <a onclick="yuiMessage('警告哦，这是一条警告消息',{type:'warning',effect:'dark'})" yui-button>警告</a>

        <a onclick="yui.message.danger('错了哦，这是一条错误消息',{effect:'dark'})" yui-button>危险</a>
