Dialog 对话框
在保留当前页面状态的情况下，告知用户并承载相关操作。
基本用法
点击打开Dialog
提示
这是一段信息
确定 取消 

        <a yui-button type="text" onclick="yui.dialog.open('my-dialog')">点击打开Dialog</a>

        <div yui-dialog id="my-dialog">
            <div header>
                <div title>提示</div>
                <i class="yui-icon closed" onclick="yui.dialog.close('my-dialog')"></i>
            </div>
            <div body>这是一段信息</div>
            <div footer>
                <a yui-button type="primary" onclick="yui.dialog.close('my-dialog')">确定</a>
                <a yui-button onclick="yui.dialog.close('my-dialog')">取消</a>
            </div>
        </div>
