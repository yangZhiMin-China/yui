Notify 通知
悬浮出现在页面角落，显示全局的通知提醒消息。
基本用法
可自动关闭 不会自动关闭 

        <a yui-button onclick="yuiNotify({title:'标题名称',message:'这是一条自动关闭的消息'})">
            可自动关闭
        </a>
        <a yui-button onclick="yuiNotify({title:'标题名称',message:'这是一条不会自动关闭的消息',duration:0})">
            不会自动关闭
        </a>
带有倾向性
消息 成功 警告 危险 

        <a yui-button onclick="yui.notify.info({title:'消息',message:'这是一条消息的通知'})">
            消息
        </a>
        <a yui-button onclick="yuiNotify({title:'成功',message:'这是一条成功的通知',type:'success'})">
            成功
        </a>
        <a yui-button onclick="yui.notify.warning({title:'警告',message:'这是一条警告的通知'})">
            警告
        </a>
        <a yui-button onclick="yuiNotify({title:'危险',message:'这是一条危险的通知',type:'danger'})">
            危险
        </a>
